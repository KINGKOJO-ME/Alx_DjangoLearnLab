# LibraryProject

This project was created to gain familiarity with Django by setting up a Django development environment and creating a basic Django project.

## Objective
- Install Django
- Create a Django project named LibraryProject
- Run the Django development server
- Understand the default Django project structure

## How to Run the Project
1. Navigate into the project directory
2. Run: `python manage.py runserver`
3. Open your browser and go to http://127.0.0.1:8000/
